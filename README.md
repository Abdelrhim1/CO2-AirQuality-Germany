# CO₂ & Air Quality Analysis in Germany (2000–2024)

## Project Overview
This project analyzes long-term trends in CO₂ emissions and air quality indicators in Germany.
It demonstrates an end-to-end data analytics workflow, combining a Geoscience background with
modern data analytics tools.

## Objectives
- Analyze historical CO₂ emission trends in Germany
- Explore air quality indicators (PM2.5, NO₂)
- Identify relationships between emissions and air quality
- Present insights using an interactive Power BI dashboard

## Data Sources
- Our World in Data (CO₂ emissions)
- European Environment Agency (Air Quality)

## Tools
- Power BI
- Python (pandas)
- SQL
- Excel

## Status
Work in progress — data cleaning and dashboard development ongoing.
